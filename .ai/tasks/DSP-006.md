# DSP-006: Player Engine and UI

## Objective

Build the player engine with state machine, media renderers, and UI components.

## Scope

- PlayerStateMachine with states: IDLE, LOADING, PLAYING, PAUSED, ERROR, TRANSITIONING
- PlayerEngine for playback orchestration
- MediaRenderer base class with ImageRenderer and VideoRenderer
- TransitionManager for CSS transitions
- UI components: PlayerContainer, LoadingScreen, ErrorScreen, StatusOverlay
- Application integration with Vue 3

## Deliverables

- src/application/player/service.ts
- src/application/playlist/service.ts
- src/infrastructure/mqtt/client.ts
- src/infrastructure/storage/service.ts
- src/infrastructure/http/\* (Axios-based services)
- src/infrastructure/platform/\* (Tizen/Web adapters)
- src/presentation/components/\*.vue
- src/App.vue
- src/main.ts
