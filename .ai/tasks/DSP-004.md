# DSP-004: MQTT Layer and Commands

## Objective
Implement MQTT client with connection management, command handling, and event publishing.

## Scope
- MQTT configuration and connection manager with exponential backoff
- MqttClient implementation of IMqttRepository
- CommandHandler with validation and deduplication
- Command implementations: reload_playlist, restart_player, play, pause, set_volume, screenshot
- EventPublisher for status, events, and logs
- Screenshot command with base64 response

- src/infrastructure/mqtt/MqttClient.ts
