# DSP-005: Platform Abstraction

## Objective
Create platform abstraction layer for Tizen, WebOS, and Web platforms.

## Scope
- PlatformAdapter for runtime detection
- AbstractPlatform base class
- TizenPlatform with webapis.avplay
- WebosPlatform with HTML5 video and LS2 API
- WebPlatform for development/testing

## Deliverables
- src/infrastructure/platform/PlatformAdapter.ts
- src/infrastructure/platform/AbstractPlatform.ts
- src/infrastructure/platform/TizenPlatform.ts
- src/infrastructure/platform/WebosPlatform.ts
- src/infrastructure/platform/WebPlatform.ts

## Dependencies
DSP-003, DSP-004

## Time Estimate
10 hours

## Acceptance Criteria
- [ ] Platform auto-detected at runtime
- [ ] Tizen: AVPlay API for media
- [ ] WebOS: HTML5 video + Luna services
- [ ] Web: HTML5 video for dev
- [ ] Common interface across platforms
- [ ] Screenshot mock for platforms without API
